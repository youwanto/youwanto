# CLAUDE.md

## Project
- This is a Next.js + TypeScript ecommerce project deployed on Vercel.
- The site is an English fashion ecommerce experience.
- Priorities: correctness, maintainability, conversion, performance, SEO.

## Workflow
- For non-trivial tasks: explore first, then plan, then implement, then verify.
- Read relevant docs and code before making claims.
- Do not over-engineer.
- Keep changes small and reviewable.
- Flag any likely impact on SEO, routing, search, analytics, or checkout.

## Code
- Prefer existing project patterns.
- Keep business logic out of page components when possible.
- Preserve metadata and route stability.
- Use clear ecommerce naming.

## Verification
- Always define verification steps.
- Do not claim success without verification.
